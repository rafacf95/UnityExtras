# UnityExtras

Este repositório é um projeto Unity criado como parete do meu portifío pessoal. Seu principal objetivo é desmontrar habilidades práticas na costumização do editor da Engine.

Este repositório demonstra design modular, boas práticas de código e soluções práticas para problemas comuns no desenvolvimento com Unity.
Desenvolvido durante as aulas da plataforma EBAC.

## Visão Geral

Projetos Unity frequentemente exigem pequenos utilitários que são reutilizados em diferentes jogos ou aplicações. O **UnityExtras** centraliza essas ferramentas em um único repositório, facilitando sua manutenção e integração em outros projetos.

### Exemplos de Conteúdo

- Ferramentas de editor para acelerar o desenvolvimento
- Utilitários de runtime e classes auxiliares
- Scripts comuns de gameplay e sistemas
- Extensões para componentes nativos da Unity

> As funcionalidades disponíveis dependem da implementação atual dentro da pasta `Assets`.

## Tecnologias e Ferramentas

- **Engine:** Unity
- **Linguagem:** C#
- **Controle de Versão:** Git
- **Editor:** Unity Hub / Unity Editor

## Como Começar

1. Clone o repositório:

   ```bash
   git clone https://github.com/rafacf95/UnityExtras.git
   ```

2. Acesse o diretório do projeto:

   ```bash
   cd UnityExtras
   ```

3. Abra o projeto no **Unity Hub** utilizando uma versão compatível da Unity.

4. Explore os utilitários disponíveis na pasta `Assets`.

# Casos de Uso

Este projeto é indicado para:

- Demonstração em portfólio
- Reutilização de utilitários Unity em diferentes projetos
- Referência de padrões de scripts e ferramentas de editor
